Kites FTP Client

A modern, cross-platform desktop FTP client with multi-connection support, built with Electron.

Download Get Started View on GitHub

Features

Multiple Connections

Connect to multiple FTP servers simultaneously and switch between them with tabs.

File Explorer

Explorer-like interface with directory tree and file listing for intuitive navigation.

File Operations

Upload, download, create, edit, and delete files and directories with ease.

Context Menus

Right-click context menus for quick access to common operations.

Custom Editor

Use your preferred text editor for editing remote files directly.

Dark Mode

Toggle between light and dark themes for comfortable viewing.

Connection Management

Save, organize, and search your FTP connections for quick access.

Cross-Platform

Works on Windows, macOS, and Linux for a consistent experience.

Installation

Download

Download the latest release for your platform from the Releases page.

Build from Source

# Clone this repository
git clone https://github.com/kites-dev/Kites-FTP.git

# Navigate to the repository
cd kites-ftp-client

# Install dependencies
npm install

# Run the app in development mode
npm start

# Build for your platform
npm run dist
