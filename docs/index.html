Getting Started

Easy installation options for all platforms

Download and Install

Download the latest release for your platform from our Releases page or use the download buttons below.

Build from Source

For developers who want to build from source or contribute to the project:

# Clone this repository
git clone https://github.com/kites-dev/Kites-FTP.git

# Navigate to the repository
cd kites-ftp-client

# Install dependencies
npm install

# Run the app in development mode
npm start

# Build for your platform
npm run dist

System Requirements

  • Windows 10 or later (64-bit)
  • macOS 10.13 or later
  • Linux (Ubuntu, Debian, Fedora, or other major distributions)
  • 4GB RAM minimum, 8GB recommended
  • 100MB free disk space
